What is WinPDFprint

WinPDFprint is a free Windows application (developed with PERL and Tk) that permits you to print PDF files via Acrobat reader.
WinPDFprint use

WinPDFprint is composed of 6 buttons and a listbox.
  • Quit to close the application
  • Select File through a 'file select' window you can select only PDF files (the control is made on the file extension .pdf and on the presence of %PDF in the first line of thes selected file). After the selection you see in the list box your selected files.
    Remember to select files in a different share (e.g. V:) you need to write inside Directory fild the share (e.g. V:/*.pdf)
  • Print First Pass the first selected file in the list to Acrobat reader to print it and remove it from the listobox.
  • Unselect All to clean the listbox
  • About
  • Help
You can remove only one entry form listbox with a double click with the left button of the mouse on it.

Limitation
In this first release the software look for Acrobat reader in this location
C:\\Acrobat3\\Reader\\acrord32.exe
